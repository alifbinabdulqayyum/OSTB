# _One Stone Three Birds_: Three-Dimensional Implicit Neural Network for Compression and Continuous Representation of Multi-Altitude Climate Data

## Introduction
This is the code repository of the paper: _One Stone Three Birds_: Three-Dimensional Implicit Neural Network for Compression and Continuous Representation of Multi-Altitude Climate Data
<!-- [Implicit Neural Representations for Simultaneous Reduction and Continuous Reconstruction of Multi-Altitude Climate Data
](https://doi.org/10.1109/MLSP58920.2024.10734742) -->


## Data
To access the data, explore [Wind Integration National Dataset (WIND) Toolkit Data Download](https://github.com/NREL/hsds-examples.git)

To download the data, run the following commands:
```
bash data.sh
```

## Train
Run the following commands:
```
bash train.sh
```

## Test
Run the following commands:
```
bash test.sh
```
